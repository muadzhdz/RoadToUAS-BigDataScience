# Sumber Dataset — Proyek UAS Big Data Science

Semua dataset disimpan di: **`./datasets/`**

---

## 1. Retail — Supermarket Sales
- **File:** `retail/supermarket_sales.csv`
- **Sumber:** GitHub — Vicky5697/Supermarket-Sales-Data-Analysis
- **URL:** https://raw.githubusercontent.com/Vicky5697/Supermarket-Sales-Data-Analysis/main/Supermarket_Sales.csv
- **Deskripsi:** Data transaksi supermarket fiktif dari 3 cabang (Yangon, Naypyitaw, Mandalay) selama 3 bulan (Jan-Mar 2019). 1000 baris.
- **Kolom kunci:** `Date`, `Product line`, `City`, `Total`, `Quantity`, `Rating`

---

## 2. Health — Healthcare Dataset
- **File:** `health/healthcare_dataset.csv`
- **Sumber:** GitHub — yuanz25/healthcare-data-analysis (data asli dari Kaggle: prasad22/healthcare-dataset)
- **URL:** https://github.com/yuanz25/healthcare-data-analysis/raw/main/healthcare_dataset.csv
- **Deskripsi:** Data sintetis pasien rumah sakit mencakup 55.500 rekam medis dengan informasi admisi, diagnosis, tagihan, dan hasil tes.
- **Kolom kunci:** `Date of Admission`, `Medical Condition`, `Gender`, `Insurance Provider`, `Billing Amount`, `Age`

---

## 3. Environment — Beijing PM2.5
- **File:** `environment/beijing_pm25.csv`
- **Sumber:** GitHub — jbrownlee/Datasets (UCI Machine Learning Repository)
- **URL:** https://raw.githubusercontent.com/jbrownlee/Datasets/master/pollution.csv
- **Deskripsi:** Data kualitas udara Beijing dari 2010-2014 dengan data polusi PM2.5 per jam. 43.825 baris.
- **Kolom kunci:** `year`, `month`, `day`, `hour`, `pm2.5`, `TEMP`, `PRES`, `cbwd` (arah angin)

---

## 4. Finance — Multi Stock Data
- **File:** `finance/stock_data.csv`
- **Sumber:** GitHub — plotly/datasets
- **URL:** https://raw.githubusercontent.com/plotly/datasets/master/stockdata.csv
- **Deskripsi:** Data harga saham harian 5 perusahaan (MSFT, IBM, SBUX, AAPL, GSPC) dari 2007-2016. 2.307 baris.
- **Kolom kunci:** `Date`, ticker saham (MSFT, IBM, SBUX, AAPL, GSPC), harga penutupan

---

## 5. Agriculture/Food — Coffee Quality Ratings
- **File:** `agriculture/coffee.csv`
- **Sumber:** GitHub — rfordatascience/tidytuesday (data dari Coffee Quality Institute)
- **URL:** https://raw.githubusercontent.com/rfordatascience/tidytuesday/master/data/2020/2020-07-07/coffee_ratings.csv
- **Deskripsi:** Data 1.339 sampel kopi dari berbagai negara, dinilai oleh Coffee Quality Institute. Mencakup spesies, asal, metode processing, dan skor rasa.
- **Kolom kunci:** `grading_date`, `species`, `country_of_origin`, `variety`, `total_cup_points`, `aroma`, `flavor`, `altitude_mean_meters`
